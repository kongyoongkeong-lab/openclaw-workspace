# OpenClaw Shortcut Registry

| Shortcut | Action |
|---|---|
| `指令列表` | Show this shortcut registry. |
| `今日新闻` / `每日新闻` / `daily news` | Run the daily news workflow with cache-first behavior. |
| `今日AI` / `今日 AI` / `AI日报` / `最新AI动态` | Run the latest AI news workflow. |
| `额度查询` | Run the Codex/OpenAI quota query without exposing tokens. |
| `额度切换` / `Codex切换` / `账号切换` | Reorder Codex auth profiles by quota without exposing tokens. |
| `塔维切换` / `Tavily切换` / `搜索切换` / `塔维账号` | Check Tavily accounts and choose the available search key without exposing secrets. |
| `API模型` | Set workspace-local model routing to API-first. |
| `本地模型` | Set workspace-local model routing to local-first. |
| `混合模型` | Set workspace-local model routing to hybrid. |
| `当前模型` | Show active model, fallbacks, quota, and routing status. |
| `模型策略` | Summarize `docs/model_runtime_strategy.md`. |
| `架构模式` | Set protocol mode to `architect_mode`. |
| `当前模式` | Show current protocol mode. |
| `稳定模式` | Set protocol mode to `stable`. |
