# Video Workflow Templates

Git 跟踪的 ComfyUI 视频工作流模板。

## Available Templates
| File | Model | Mode | Quality |
|------|-------|------|---------|
| `txt2vid_wan2.2.json` | Wan 2.2 | Text-to-Video | ⭐⭐⭐ |
| `txt2vid_ltx2.3.json` | LTX 2.3 | Text-to-Video | ⭐⭐ |
| `img2vid_wan2.2.json` | Wan 2.2 | Image-to-Video | ⭐⭐⭐ |
| `img2vid_ltx2.3.json` | LTX 2.3 | Image-to-Video | ⭐⭐ |

## Export from ComfyUI
1. Open ComfyUI at http://localhost:8188
2. Build workflow using blueprint: File → Open → `blueprints/Text to Video (Wan 2.2).json`
3. File → Export (API format)
4. Save to this directory
5. Commit to GitHub
