# 📋 快捷指令列表

## 📰 新闻

| 输入 | 操作 |
|------|------|
| `今日新闻` | 生成大马+全球新闻简报（缓存6h） |
| `今天有什么新闻` | 同上 |
| `马来西亚新闻` | 同上 |
| `每日新闻` / `daily news` | 同上 |
| `刷新新闻` | 强制重新抓取今日新闻 |
| `重新抓取` | 同上 |
| `重新生成今日新闻` | 同上 |
| `deep search` / `深度搜索` | 深度搜索模式（Tavily不限量） |

## 🤖 AI 资讯

| 输入 | 操作 |
|------|------|
| `今日AI` | 生成近7天全球AI日报（缓存7天） |
| `AI日报` | 同上 |
| `最新AI动态` | 同上 |
| `daily AI` / `AI news` | 同上 |

## 🗂️ 文件 / 存档

| 输入 | 操作 |
|------|------|
| `指令列表` | 显示此清单 |

## 📁 归档路径

| 内容 | 路径 |
|------|------|
| 每日新闻 | `news/YYYY-MM-DD-daily-news.md` |
| AI 日报 | `news/ai/YYYY-MM-DD-daily-ai.md` |
| 工作流 | `workflows/daily_news.md` |
| 工作流 | `workflows/daily_ai.md` |

## 🤖 系统 / 维护

| 输入 | 操作 |
|------|------|
| `/status` | 查看 session 状态 |
| `/reasoning` | 切换思维链显示 |
