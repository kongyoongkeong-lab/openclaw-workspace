# 📋 四字指令表

> 规则：常用提示词统一为 **四个华语字**，方便记忆和输入。

## 📰 新闻

| 四字指令 | 用途 | 备注 |
|---|---|---|
| `今日新闻` | 大马 + 全球新闻简报 | 缓存 6 小时 |
| `大马新闻` | 偏重大马新闻 | 缓存 6 小时 |
| `每日新闻` | 同今日新闻 | 缓存 6 小时 |
| `刷新新闻` | 强制重新抓取 | 跳过缓存 |
| `深度搜索` | 深度新闻搜索 | 更完整但较慢 |

## 🤖 AI 资讯

| 四字指令 | 用途 | 备注 |
|---|---|---|
| `智能日报` | 全球 AI 日报 | 缓存 7 天 |
| `今日智能` | 同智能日报 | 缓存 7 天 |
| `最新智能` | 最新 AI 动态 | 缓存 7 天 |
| `模型动态` | 模型 / Agent 新闻 | AI 子分类 |
| `算力芯片` | 芯片 / 算力 / 数据中心 | AI 子分类 |

## 🎬 媒体

| 四字指令 | 用途 | 当前能力 |
|---|---|---|
| `生成视频` | 文本生成视频 | Google Veo |
| `图片视频` | 图片转视频 | 需附图 |
| `视频能力` | 查看视频 API / 能力 | 显示 provider 状态 |

## 🧠 记忆 / 回顾

| 四字指令 | 用途 | 备注 |
|---|---|---|
| `昨日回顾` | 回顾昨天工作 | 查 memory/git/session |
| `今日回顾` | 回顾今天工作 | 查 memory/git/session |
| `当前进展` | 查看当前任务进展 | 简短状态汇报 |
| `团队状态` | 查看可用 agents | 显示 runtime agents |

## 🛠️ 系统

| 四字指令 | 用途 | 备注 |
|---|---|---|
| `指令列表` | 显示此清单 | 四字表格版 |
| `会话状态` | 查看 session 状态 | 等同 `/status` |
| `思维切换` | 切换 reasoning 显示 | 等同 `/reasoning` |
| `协议审查` | 检查 protocol / GitHub issue | 系统维护 |

## 📁 归档路径

| 内容 | 路径 |
|---|---|
| 每日新闻 | `news/YYYY-MM-DD-daily-news.md` |
| AI 日报 | `news/ai/YYYY-MM-DD-daily-ai.md` |
| 新闻工作流 | `workflows/daily_news.md` |
| AI 工作流 | `workflows/daily_ai.md` |
| 指令列表 | `workflows/指令列表.md` |
